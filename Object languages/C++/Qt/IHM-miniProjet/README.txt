/// V.1.0:						
///	-Fichier: 					
///		-Ouvrir : Selectionner et lire un txt.		
///		-Sauvegarder sous : Sauvegarde le texte 		
///			affiché dans le dossier de		
///			votre choix.			
///		-Sauvegarder : Sauvegarder le texte dans le dossier	
///			de sauvegarde précédent.		
///		-Quitter : Quitte le programme.		
///		- [Path/fichier] représente l'accés au dernier fichier	
///			chargé dans TextFinder, peut être	
///			chargé a nouveau en cliquant dessus	
///						
///	UTILISATION:				
///		-Ouvrir un txt				
///		-Indiquer le mot a rechercher 		
///		-Recherche :				
///			-Effectue une recherche globale du mot	
///			recherché.			
///		-Suivant :				
///			-Effectue une recherche		
///				mot par mot.
///		-Précédent :
///			-Effectue une recherche sur le mot
///				recherché précedement.		
///		-Reset :				
///			- Reinitialise la fenetre		
///		Mot complet :				
///			- Indique si la recherche doit être	
///			effectuée sur un mot complet ou sur	
///			une suite de lettres.		
///						
///	FONCTIONNALITES:				
///		- Drag&Drop de texte dans la fenetre autorisé.	
///		- Affiche le numero de ligne et de lettre dans la ligne	
///			de la position du curseur.		
///		- Affiche le nombre d'occurences de la recherche.	
///		- Affiche a quelle occurence de la recherche le curseur	
///			est positionné.		
///		- Redimension de la fenetre et adapation des widgets 	
///			autorisé.			
///		- Recherche occurence de mots ou de suite de	
///			charactéres.			
///		- Charger un fichier texte.			
///		- Modifier et sauvegarder un document en txt.	
///		- Quitter depuis la barre de menu.		
///		- Reset de la fenetre possible		
///		- Affichage de messages d'erreurs et informations:	
///			- Sauvegarde		
///			- Fin de recherche		
///			- Reset			
///			- Mot manquant		
///			- Mot inconnu			
///		- Sauvegarde dans le répertoire courant		
///		- Enregistre le dernier fichier chargé, celui si peut être	
///			retrouvé dans 'Fichier', limité aux 5 	
///			derniers fichiers.	
///		- Revenir en arriére dans la recherche avec le bouton
///			précédent.
///		- Controle des boutons si la recherche débute ou se 
///			termine.
///						
///
